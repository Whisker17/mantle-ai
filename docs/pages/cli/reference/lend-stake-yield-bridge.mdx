## `lend markets`

- Status: Implemented
- Syntax: `mantle lend markets [--asset <asset>] [--protocol <lendle|aurelius|aave_v3|all>]`
- Defaults: `--protocol all`
- Cache TTL: `5m`

```bash
./mantle lend markets --protocol all --results-only
```

## `bridge quote`

- Status: Implemented
- Syntax:

```bash
mantle bridge quote --from-chain <caip2> [--to-chain <caip2>] --asset <symbol|address> --amount <decimal> [--provider <official|across|best>]
```

- Required flags: `--asset`, `--amount`
- Defaults: `--from-chain eip155:1`, `--provider best`
- If `--to-chain` omitted, defaults to active Mantle network (`eip155:5000` or `eip155:5003`).
- Cache TTL: `30s`

```bash
./mantle bridge quote --from-chain eip155:1 --to-chain eip155:5000 --asset USDC --amount 100 --provider best --results-only
```
