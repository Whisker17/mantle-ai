# Mantle AI Documentation

This site contains product documentation for the Mantle AI monorepo.

## Documentation Areas

- [CLI](/cli): Agent-first Mantle retrieval CLI docs.

## Repository Layout

- `docs/`: Nextra documentation website implementation.
- `specs/plans/`: design and implementation planning documents.
- `cli/`: Go CLI source.

## Local Development

```bash
npm --prefix docs install
npm --prefix docs run dev
```

Open `http://localhost:3000`.
